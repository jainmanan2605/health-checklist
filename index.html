Connect your GitHub account

Your check-ins are stored in a private GitHub Gist in your account, so they sync across all your devices. One-time setup per device:

  1. Open github.com/settings/tokens/new (pre-filled for you)
  2. Set expiration to No expiration (or your preference)
  3. Make sure only the gist scope is ticked
  4. Click Generate token and copy it

The token is kept only in this browser and is used solely to read/write one private gist. It cannot touch your repositories.

✨ All done — great job!

Your habits

Rename habits by typing directly in the box. Each habit can be daily, a weekly target (e.g. 3×/week), or tied to specific weekdays. Deleting a habit keeps its past history in old reports.
